### To work with hexo

```npm install -g hexo-cli```

After that: https://hexo.io/docs/setup.html

```
$ hexo init <folder>
$ cd <folder>
$ npm install
```

Main documentation: https://hexo.io/docs/

Configuration: https://hexo.io/docs/configuration.html
